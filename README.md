## üöÄ **Deployment Guide**

### **Step 1: Upload to GitHub**

1. **Create GitHub Account** (if you don't have one):
   - Go to [GitHub.com](https://github.com)
   - Click "Sign up" and create your account

2. **Create New Repository**:
   - Click the "+" icon in top right corner
   - Select "New repository"
   - Repository name: `insurance-management-system`
   - Make it **Public** (required for free Streamlit hosting)
   - Check "Add a README file"
   - Click "Create repository"

3. **Upload Files**:
   - In your new repository, click "uploading an existing file"
   - Drag and drop these files:
     - `app.py` (main application)
     - `requirements.txt` (Python dependencies)
   - Add commit message: "Initial insurance management system"
   - Click "Commit changes"

### **Step 2: Deploy on Streamlit Cloud**

1. **Visit Streamlit Cloud**:
   - Go to [share.streamlit.io](https://share.streamlit.io)
   - Click "Sign up" and use your GitHub account

2. **Deploy Application**:
   - Click "New app"
   - Select your GitHub repository: `insurance-management-system`
   - Main file path: `app.py`
   - Click "Deploy!"

3. **Wait for Deployment**:
   - Initial deployment takes 2-5 minutes
   - You'll see build logs and progress
   - Once complete, you'll get a live URL

### **Step 3: Access Your Application**

Your insurance management system will be live at:
`https://[your-username]-insurance-management-system-app-[random-string].streamlit.app`
